Cards

This component is used to present blog posts for different photo sessions. Users are able to browse through the cards and read the session title and date. An image depicting the showcased session is displayed on each card.




Accessibility

When using a card, be sure to include descriptive labels for <p> elements and headers. This helps assistive technologies identify these components. In addition, be sure to provide alt text for any images.

Details

A total of four cards will be featured on the home page. These will represent different blog posts and provide information about the photo sessions. Users are able to read more about a session and view the full gallery by clicking on a card.

Code

              
            <div class="card" style="width: 18rem;">
              <img src="..." class="card-img-top" alt="...">
              <div class="card-body">
                <p class="card-text">Some quick example text to build on the card title and make up the bulk of the card's content.</p>
              </div>
            </div>
