Session Info

This component is used to display session information. Users are able to scroll through and read about the details including the session name, category, and date. The information also features a few paragraph elements that describe the session.




Accessibility

When using this component, be sure to include descriptive labels for <p> elements and headers. This helps assistive technologies with identification.

Details

The session information will be featured at the top of each blog page. It will provide more detailed information about the photo sessions for the users to read.

Code

              
            <div class="col-md-8">
              <div class="session-body">
                <h2 class="session-title">Session title</h2>
                <p class="session-text"><small class="text-muted">Session date</small></p>
                <p class="session-text"><small class="text-muted">Session category</small></p>
                <p class="session-text">Session information</p>
              </div>
            </div>
